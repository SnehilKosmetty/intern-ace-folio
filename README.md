# PallaviNile Portfolio

A personal portfolio website built with React + TypeScript. It uses Tailwind CSS for styling and Framer Motion for subtle animations. The codebase includes a responsive navigation component (`src/components/Navbar.tsx`) and other typical portfolio sections (About, Experience, Projects, Skills, Contact).

## Features
- Responsive layout (desktop + mobile)
- Animated navigation and UI using `framer-motion`
- Tailwind CSS utility classes
- Iconography via `lucide-react`
- Reusable UI primitives (example: `src/components/ui/button`)
- Resume download link (check `public/assets` or `public/resume.pdf`)

## Tech stack
- TypeScript
- React (project structure uses `.tsx` files and path alias `@/`)
- Tailwind CSS
- Framer Motion
- lucide-react

## Prerequisites
- Node.js (16+ recommended, 18+ preferred)
- npm, Yarn, or pnpm
